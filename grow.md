# Улучшение проекта
## Сильные стороны конкурентов
- Сквот - агрессивная эстетика, скейтерский уличный стиль
- Траектория - весь сайт галерея товаров

## Что можно улучшить
1. Галерея товаров
   - Возможность пролистывать несколько фотографий одного товара. Это помогает лучше рассмотреть товар перед покупкой. Добавить в карточку товара возможность листать 2–3 фотографии, а также добавить блок фотогалереи для визуала и разбавления контента.
2. Отзывы о товарах
   - С наличием отзывов доверие у клиента возрастает. Добавить блок с отзывами покупателей.
3. Подписка на новостную рассылку
   - Пользователь заходит, смотрит, уходит. Нет способа напомнить о себе, сообщить о скидках или новых поступлениях. Добавить форму подписки "Подпишись и получи скидку 5% на первый заказ". Это поможет собирать базу клиентов.
4. Преимущества нашего скейтшопа
   - Добавить блок с преимуществами, например у нас есть Оплата частями (Сплит, Долями), бесплатная доставка от N₽.
