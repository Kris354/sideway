# Оптимизация
- Для чего нужна оптимизация?
  - Оптимизация нужна для для быстрой загрузки сайта. Tсли сайт тормозит, то пользователь уйдет к конкурентам. Адапптация под разные устройства, комфорт и приятное взаимодействие у пользователя.
- Что будем оптимизировать?
  - Изображения - сжать изображения, но без потери качества
  - Убрать лишний код
  - Кеширование - при повторных посещениях сайта брайзер берет файлы из кеша
- Как будем оптимизировать?
  - Использовать современные форматы для сжатия PNG - TinyPNG, картинки товаров переведем в WebP
  - Кешируем изображения, шрифты и стили, скрипты
- Сравнительные результаты
  - Время загрузки до 4–5 секунд	после 1.5–2 секунды
  - Вес страницы	до ~3 MB	после ~1 MB
  - Все файлы при повторном посещении берутся из кеша
На выходе получаем быстрый и удобный сайт. При первом заходе все грузится почти в два раза быстрее, а при повторных визитах страница открывается практически мгновенно за счет кеширования.
