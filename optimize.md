# Оптимизация
- Для чего нужна оптимизация?
  - Оптимизация нужна для для быстрой загрузки сайта, (если сайт тормозит, то пользователь уйдет к конкурентам), адапптация под разные устройства, комфорт и приятное взаимодействие у пользователя.
- Что нужно оптимизировать?
  - Изображения (если выгружать картинки прямо с камеры, сайт умрет)
  - Фильтры и каталог
  - Мобильная версия
- Как будем оптимизировать?
  - Использовать современные форматы для сжатия изображения, чтобы фото загружались только тогда, когда пользователь долистал до них.
  - Оптимизировать запросы к базе данных
  - Сделать мобильную версию удобнее
