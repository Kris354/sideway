# Оптимизация
- Для чего нужна оптимизация?
  Оптимизация нужна для для удержания клиентов (если сайт тормозит, то пользователь уйдет к конкурентам), медленная загрузка страницы товара или корзины убивает продажи. Оптимизация фильтров и поиска = больше продаж.
- Что нужно оптимизировать?
  1. Изображения (если выгружать картинки прямо с камеры, сайт умрет)
  2. Фильтры и каталог
  3. Мобильная версия
- Как будем оптимизировать?
  1. Использовать современные форматы для сжатия изображения, чтобы фото загружались только тогда, когда пользователь долистал до них.
  2.  Оптимизировать запросы к базе данных
  3.  Сделать мобильную версию удобнее
