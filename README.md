# Одностраничный лендинг для скейтшопа SIDEWAY
Проект представляет собой современный лендинг для скейтшопа, разработанный на React. Сайт выполнен в современном спортивном и минималистичном стиле с использованием темно-синего, черного и белого цветов. Задача сайта показать товары, рассказать о компании и побудить польователя перейти в каталог.

## Запуск проекта
1. Скачать проект с GitHub git clone https://github.com/Kris354/sideway.git или скачать ZIP-архив с репозитория и распаковать.
2. Перейти в папку проекта cd sideway.
3. Установить зависимости npm install.
4. Запустить проект в режиме разработки npm start. После этого проект автоматически откроется в браузере.

## Структура сайта
- **Шапка** — логотип, бургер-меню, иконки (поиск, корзина, избранное, профиль)
- **Hero-блок** — фотография на весь экран, текст с позиционированием бренда, крупная кнопка призыва к действию
- **Новинки** — три карточки товаров с изображениями и кнопкой "Смотреть"
- **Популярное** — аналогичный блок с тремя карточками
- **О нас** — текстовый блок с описанием преимуществ компании
- **Футер** — логотип, иконки, контактные телефоны, ссылки на соцсети
