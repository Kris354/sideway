# Одностраничный лендинг для скейтшопа SIDEWAY
Проект представляет собой современный лендинг для скейтшопа, разработанный на React. Сайт выполнен в современном спортивном и минималистичном стиле с использованием темно-синего, черного и белого цветов. Задача сайта показать товары, рассказать о компании и побудить польователя перейти в каталог.

## Запуск проекта
1. Скачать проект с GitHub `git clone https://github.com/Kris354/sideway.git` или скачать ZIP-архив с репозитория и распаковать
2. Перейти в папку проекта `cd sideway`
3. Установить зависимости `npm install`
4. Запустить проект в режиме разработки `npm start`. После этого проект автоматически откроется в браузере

## Структура сайта
- **Header** - бургер-меню, логотип, иконки (поиск, корзина, избранное, профиль)
- **Hero-блок** - фотовое изображение, слоган магазина, кнопка "Перейти в каталог" призывающая к действию
- **Новинки** - карточки товаров с кнопкой "Смотреть" призывающей к действию
- **Популярное** - карточки товаров с кнопкой "Смотреть" призывающей к действию
- **О нас** - информация о компании
- **Footer** - логотип, иконки возможности оплаты, контактные телефоны, ссылки на соцсети

## Стили и дизайн
- **Цветовая гамма:** темно-синий (#03116C), белый, черный
- **Шрифты:** тонкий (200) для основного текста, жирный (800) для заголовков и логотипа
- **UI:** в современном спортивном стиле, навигация без лишних деталей.
- **Адаптивность:** сайт корректно отображается на десктопах, планшетах и мобильных устройствах
