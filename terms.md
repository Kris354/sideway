# ТЗ разработки сайта бордшопа SIDEWAY
# Введение
Данное техническое задание описывает требования по разработке сайта интернет-магазина бордов SIDEWAY.

# Общие сведения
1. Назначение системы: сайт создается для
  - Продажи товаров, приема и обработки заказов на скейты, сноуборды и серфборды с помощью онлайн-оплаты
  - Информирование покупателей размещением полезной информации (характеристики, гайд для новичков)
  - Круглосуточный доступ чтобы клиент мог зайти, посмотреть ассортимент и оформить заказ в любое время.

2. Основные конкурентные преимущества компании:
   - Рассрочка для новичков: подключены Яндекс Сплит и Долями — можно купить доску, даже если бюджет ограничен
   - Понятный статус заказа: клиент видит в личном кабинете, на каком этапе его покупка (оформлен, отправлен, в пути)
   - Экспертность (помогаем выбрать наилучший вариант)
   - Быстрая и всегда доступная поддержка, поможем со всеми вопросами, с выбором модели и размера.
  
3. Целевая аудитория
   - Новички 17–25 лет, доход средний, часто ограничен: покупают первую доску, не особо разбираются
   - Любители 20–30 лет, доход средний: уверенно катаются, следят за новинками, готовы обновлять снаряжение. Понимают чего хотят
   - Опытные 25–35 лет, доход выше среднего: ценят качество, сервис и премиальные бренды. Могут заказывать дорогие модели.

# Структура сайта
- Шапка - бургер-меню с категориями, логотип при нажатии на который ты возвращаешься на начало главной страницы, иконки в ряд (лупа, избранное, корзина и профиль)
- Блоки - херо-блок - фотография, заголовок и подзаголовок, кнопка "Перейти в каталог". Заголовок и каталог-карусель новинок и популярных товаров. Крупные блоки с категориями. О нас. Отзывы.
- Футер - логотип, информация о компании (о нас, контакты, доставка и оплата), поддержка, соцсети, иконки платежных систем.

# Описание экранов
Главная (лендинг с блоками)
- Пользователь видит логотип, фоновое фото, заголовок и подзаголовок, кнопка Перейти в каталог - при нажатии переход в каталог всех товаров, при нажатии на лого - переход в начало страницы. Нажать кнопку меню - открывается меню с категориями товара (скейтборды, сноуборды, серфборды). Нажать кнопку Профиль - перейдет в личный кабинет/форма регистрации. Нажать кнопку Лупа - вылезет поисковая строка. Нажать кнопку Избранное - переход в список сохраненных товаров. Нажать кнопку Корзина - переход в корзину. 
- Блок Новинки - карусель с карточками товаров и кнопкой Смотреть - при нажатии переход на товар.
- Блок Популярные товары - карусель с карточками товаров и кнопкой Смотреть - при нажатии переход на товар.
- Блок Категории - 3 крупных блока с картинками (скейты, сноуборды, серфборды) на каждом блоке кнопка Смотреть - при нажатии переход в указанную категорию.
- Блок О нас - информация про бордшоп
- Блок Отзывы - 2-3 карточки с отзывами клиентов
- Футер - Логотип, Информация о нас, Контакты для связи, Поддержка - строка для ввода текста и кнопка Отправить - при нажатии на строку вводишь текст, свою почту и нажимаешь кнопку Отправить, Соцсети, Иконки платежных систем.
